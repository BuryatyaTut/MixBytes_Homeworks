
# Instructions to Run Tests

1. **Initiate Hardhat Project**
   ```bash
   npx hardhat
   ```

2. **Download OpenZeppelin Contracts**
   ```bash
   npm install --save-dev @openzeppelin/contracts
   ```

3. **Run Tests**
   ```bash
   npx hardhat test test/Voting.ts
   ```
